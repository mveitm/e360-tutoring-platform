* which official source candidates should be considered first;

## 5. Source custody rule

Canonical PAES alignment must use official sources only.

Allowed official-source families:

* DEMRE.
* Universidad de Chile / DEMRE-hosted portals.
* Mineduc / Unidad de Currículum y Evaluación when directly relevant.
* Acceso Educación Superior when directly relevant.
* Official PDFs directly linked from official pages.

Not allowed as canonical alignment sources:

* commercial preuniversitarios;
* Wikipedia;
* media articles;
* blogs;
* AI summaries;
* SEO pages;
* copied PDFs from unofficial mirrors;
* social media excerpts unless they link back to an official source.

PDF custody rule:

* Do not download PDFs in this phase.
* Do not commit PDFs in this phase.
* Do not paste long PDF contents.
* Record only title, institution, visible date, source type, relevance, and extraction status.

## 17. Risks and mitigations

| Risk | Why it matters | Mitigation |
| --- | --- | --- |
| Using outdated source as current | M2/L1 alignment could drift from active PAES cycle. | Future extraction must verify current 2027 official pages/PDFs and supersession. |
| Treating intake as extraction | Creates false source understanding. | Label this phase as source custody only. |
| Treating source extraction as taxonomy | Temario content does not automatically define Bexauri route design. | Separate extraction, alignment, taxonomy, and content phases. |
| Copying copyrighted PDFs into repo/docs | Legal and governance risk. | No PDF downloads, no committed PDFs, no long excerpts. |
| Unofficial sources contaminate taxonomy | Weakens official PAES alignment. | Reject non-official sources for canonical alignment. |
| M2/L1 declared ready from documents alone | Public Sales-Ready drift. | Keep M2/L1 not ready and require future app/student evidence. |
| L1 text rights ignored | Reading content may create rights and editorial risk. | Require text rights/licensing and editorial review before any L1 content. |
| Example questions overused as full blueprint | Examples may not cover the whole test scope. | Treat examples as support sources, not full taxonomy. |
