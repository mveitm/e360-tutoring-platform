## 11. Revised item 2 distractor quality note

The revised distractors are plausible because they refer to objects or containers from the same original draft text.

They remain wrong for clear textual reasons:

* `Un cuaderno en buen estado` is an object category for sorting, not the item taken on arrival.
* `Una caja verde` is a container for sorted materials, not the item taken on arrival.
* `Una carpeta reutilizable` is an object category for sorting, not the item taken on arrival.

This is a distractor rationale summary only. It is not student-facing feedback.
