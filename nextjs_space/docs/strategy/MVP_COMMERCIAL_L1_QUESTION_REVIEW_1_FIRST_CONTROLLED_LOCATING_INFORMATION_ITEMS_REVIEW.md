## 6. Review method

The review checks each existing item against the accepted L1 item-quality and question-authoring boundaries:

* one explicit detail target;
* clear item wording;
* one defensible correct option;
* alternatives A-D are clear;
* distractors are plausible but wrong;
* no external knowledge;
* no inference beyond explicit information;
* no multi-detail overload unless the original text explicitly frames the detail as one action sequence;
* answer-key note remains metadata-like and not feedback;
* future feedback appears feasible.

The review uses only already-versioned documentation, the original Bexauri-created draft text, and the three authored items. No web search was used.
